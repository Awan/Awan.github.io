Encrypted Arch Linux installation

Abdullah Khabir published on 2020-01-03
What’s Arch Linux? Arch Linux is a free and open source linux distribution based on the rule Keep it Simple. Installation Arch Linux has no graphical installer. You have to install using CLI. Installation media creation Go to Downloads page and select a nearby mirror. Download the iso file. If you’re on Windows, you can use rufus to create a bootable media. On linux, you can use dd: 1 dd if=downloaded iso file of=/dev/sdX status=progress where X is your USB device’s ID.
